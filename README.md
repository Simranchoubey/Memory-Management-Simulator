# 🧠 Memory Management Simulator

🔗 **Live Demo:** https://simranchoubey.github.io/Memory-Management-Simulator/

An interactive simulator to visualize how operating systems manage memory using dynamic allocation techniques.

---

## 🚀 Features

- 📌 Add and manage processes dynamically  
- 📊 Visual representation of memory blocks  
- ⏱️ Real-time allocation & deallocation  
- 🧩 Demonstrates memory fragmentation  
- 🎯 Simple and beginner-friendly UI  

---

## 🧠 Concepts Covered

- Memory Allocation  
- Internal & External Fragmentation  
- Process Lifecycle  
- Dynamic Partitioning  

---

## ⚙️ Algorithm Used

### 🔹 Best Fit
Allocates the smallest available block that fits the process, reducing memory wastage.

---

## 🔄 How It Works

1. Enter process size and execution time  
2. Memory is allocated using Best Fit  
3. Process runs for given time  
4. Memory is freed after completion  
5. UI updates dynamically
